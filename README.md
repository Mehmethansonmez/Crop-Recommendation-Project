# 🚜 Akıllı Tarım: Mahsul Tavsiye Sistemi (Crop Recommendation)

Bu proje, **Sakarya Uygulamalı Bilimler Üniversitesi**, Elektrik-Elektronik Mühendisliği Bölümü bünyesindeki **Yapay Zekaya Giriş** dersi dönem ödevi kapsamında geliştirilmektedir. 

## 📝 Proje Özeti
Projenin amacı; topraktaki Azot (N), Fosfor (P), Potasyum (K) değerleri ile ortamın sıcaklık, nem, pH ve yağış verilerini analiz ederek, ilgili tarım arazisi için en yüksek verimi sağlayacak mahsulü (pirinç, mısır, kahve vb.) yapay zeka yöntemleriyle tahmin etmektir.

## 📅 Proje Durumu: Hafta 2 (Veri Toplama & Etiketleme)
Şu an projenin 2. haftasındayız ve aşağıdaki işlemler tamamlanmıştır:
- [x] Proje konusu ve kapsamı belirlendi.
- [x] Kaggle üzerinden 2200 satırlık ham veri seti temin edildi.
- [x] Veri envanteri raporu hazırlandı ve `/docs` klasörüne eklendi.
- [x] GitHub depo yapısı oluşturuldu.

## 📂 Klasör Yapısı
Kılavuzda belirtilen standartlara uygun olarak yapılandırılmıştır:
- **/api**: Streamlit/Flask arayüz kodları.
- **/data**: Ham ve işlenmiş veri setleri (Crop_recommendation.csv).
- **/docs**: Proje raporları ve dokümantasyon.
- **/notebooks**: Veri analizi (EDA) ve model eğitim Jüpiter dosyaları.
- **/src**: Projenin ana kaynak kodları.
